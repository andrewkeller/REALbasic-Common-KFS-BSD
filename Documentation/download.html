Acquiring this library

This library is tracked using Git, which means that there is no "central" server. Instead, there are people who each have their own publicly available clone of this repository.

With that said, there is only one currently known publicly available clone of this repository:

  • Andrew Keller's clone, hosted on GitHub

Most web interfaces for Git repositories allow you to browse history and download source code.

Downloading source code from a repository hosted on GitHub

  1. Go to the web page of the repository.
  2. (Optional) Navigate to the point in time where you would like to download source.
    1. Click on "Commits".
    2. Find and click on the commit of your choice.
  3. Click on the "Download Source" button.
  4. Click on the format of your choice.

Cloning a repository hosted on GitHub

For this action, I will assume you already know how to use Git to some extent.

As you may already have guessed, all you need to know is the URL of the repository, which is usually not the same as the URL of the web interface.

  1. Go to the web page of the repository.
  2. Right below the description of the repository, the clone URL of the repository is listed. Click on "Git Read-Only".
  3. Copy and paste the URL that appears into whatever you are using to clone the repository.
