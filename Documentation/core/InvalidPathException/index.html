InvalidPathException (Inherits from RuntimeException)

Raised when the given path is not valid.

This exception is owned by the HierarchalDictionaryFunctionsKFS module, however because this exception is so generic to begin with, it has been decided to try and keep this exception generic enough that other frameworks can use it. This is why this class' name does not end with KFS like it would normally.
