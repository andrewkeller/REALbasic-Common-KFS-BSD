BSDGlobalsKFS_FileIO (Module)

This module contains items that aid with accessing the filesystem.

Methods

FolderItem.EqualsKFS

FolderItem.EqualsKFS( other As FolderItem ) As Boolean

Returns whether or not the given FolderItems in question resolve to the same file in the local filesystem.

GetFolderItemKFS

GetFolderItemKFS( path As String ) As FolderItem

Generates a FolderItem from the given path (String), and can parse relative paths.

GetStandardAccessErrorMessageKFS

GetStandardAccessErrorMessageKFS( f As FolderItem, mustExist As Boolean, mustBeAFile As Boolean, mustBeAFolder As Boolean, mustBeReadable As Boolean, mustBeWritable As Boolean ) As String

Returns a human-readable error message for not being able to access the given FolderItem, given the provided mustExist, mustBeAFile, mustBeAFolder, mustBeReadable, and mustBeWritable properties.

FolderItem.NextSerialNameKFS

FolderItem.NextSerialNameKFS( sFileName As String, sDeliminator As String = " " ) As String

Returns a name, based on what is given, that is unique in the current directory. Optionally, delimiter can be provided between the basename and the number. The result is similar to MyFile 1.txt, MyFile 2.txt, MyFile 3.txt, and so on.

FolderItem.NextSerialNumberKFS

FolderItem.NextSerialNumberKFS( sNameStart As String, sNameEnd As String, bZeroIsBlank As Boolean = True, sLeftSide As String = " ", sRightSide As String = "", sSerialFormat As String = "0", iSerialStart As Integer = 0 ) As String

Returns the next number what would make the given file or folder name unique in the current directory, given the part of the name before the number (String), the part of the name after the number (String), whether or not the number zero should be blank (Boolean), the delimiter before the number (String), the delimiter after the number (String), the format string used to generate the number (String), and the number to start counting at (Integer).

FolderItem.ShellPathKFS

FolderItem.ShellPathKFS( bShellSpaces As Boolean = False, bForceUnixSlashes As Boolean = False ) As String

Same as FolderItem.ShellPath in RB's framework except this one optionally includes shell spaces (Boolean), and optionally forces Unix slashes on all platforms (Boolean).

Notes

FolderItem.ShellPathKFS doesn't work quite right on all platforms. It should probably be rewritten.
