DelegateClosureKFS (Inherits from Object)

This class allows you to associate arguments with a delegate.

Delegates

Note: In REALbasic, if two different delegate declarations have the same arguments and return the same type, then REALbasic treats them as literally the same. For example, if one of the NewClosure_* functions returns a delegate of type del_void_void, then you can store the result in a variable of type PlainMethodKFS or any other delegate with no arguments and no return type without the need for typecasting. It just fits right in there.

Notes

This class uses a closure to bind properties to a function pointer (a delegate). This allows a method that requires parameters to be called using a delegate that requires fewer or no parameters.
