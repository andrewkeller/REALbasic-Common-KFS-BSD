BSDGlobalsKFS_ISO (Module)

This module contains items that aid with ISO formats.

Methods

DeserializeISO8610StringAsDate

DeserializeISO8610StringAsDate( src As String, adjustToLocalTimeZone As Boolean = True ) As Date

Returns a Date object based on the given ISO 8601 string. Returns Nil if the given string does not conform to ISO 8601.

Currently, this function is only able to parse the subset of ISO 8601 used by Apple Property Lists (yyyy-mm-ddThh:mm:ssZ). A more encompassing algorithm is wanted.
