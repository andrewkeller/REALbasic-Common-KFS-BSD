DelegateTimerKFS (Inherits from Timer)

This class is a Timer that invokes a delegate in the Action event.

Constructors

[default]

New DelegateTimerKFS

Creates a new DelegateTimerKFS object with a Nil ActionMethod and mode equal to Timer.NotRunning.

Integer, PlainMethodKFS, [Boolean]

New DelegateTimerKFS( iPeriod As Integer, dActionMethod As PlainMethodKFS, bRepeat As Boolean = False )

Creates a new DelegateTimerKFS object with the given period (Integer), delegate (PlainMethodKFS), and sets it running immediately. The optional Boolean parameter specifies whether or not the timer repeats.

Properties

ActionMethod

ActionMethod As PlainMethodKFS

The delegate to invoke upon the timer firing.

Notes

Setting or unsetting the ActionMethod property does not affect the mode of the timer. In other words, setting the ActionMethod property will not start the timer, and setting the ActionMethod property to Nil will not stop the timer. The value of the ActionMethod property only affects what happens when the timer fires.
