BigStringKFS (Inherits from Object)

This class is designed to be a common base for string handling. It is not designed for string complex string manipulations.

When dealing with strings larger than 1 million bytes in length, this class will automatically swap the string data to a file on the hard drive.

Constants

kDataSourceMemory

Contains the text returned by the GetDataSourceSummary function if the data is currently located in memory.

kDataSourceMissing

Contains the text returned by the GetDataSourceSummary function if the data is located in a Folderitem than cannot be accessed.

kDataSourceStream

Contains the text returned by the GetDataSourceSummary function if the data is located in a BinaryStream.

kSwapThreshold

kSwapThreshold = 1000000

The minimum string length that causes this class to automatically swap the string to the disk. Currently set at 1 million bytes.

Length As UInt64

MemoryBlockValue As MemoryBlock

Returns or sets the value of the string data in this instance as a MemoryBlock. The result is always unlinked from the storage of this instance, except when you provided a MemoryBlock as the source of the data, in which case the same MemoryBlock object is returned.

StringDataInvolvesBinaryStream

StringDataInvolvesBinaryStream As Boolean

Tells whether or not the string data in this instance is stored in something on the other end of a BinaryStream. This does not include data structures where the back end of the BinaryStream is accounted for, such as a mutable string.

StringDataInvolvesMemoryBlock As Boolean

Tells whether or not the string data in this instance is stored in some kind of a memory block, including a MemoryBlock, a mutable string, or a String.

StringDataInvolvesRealFile

Tells whether or not the string data in this instance is stored in a file on the hard drive.

Methods

Clear

Clear()

Clears the string data in this class without destroying this instance.

Consolidate

Consolidate()

Copies the string data into an internal data structure. Similar in ideology to the Consolidate operation in iTunes.

ForceStringDataToDisk

ForceStringDataToDisk()

Relocates the storage of the string data to a swap file if it is not already somewhere on the hard drive.

ForceStringDataToMemory

ForceStringDataToMemory()

Relocates the storage of the string data to a mutable string if it is not already somewhere in memory.

GetFolderitemAccess

GetFolderitemAccess( allowSwapAccess As Boolean = True ) As Folderitem

Returns the Folderitem currently being used to contain the string data in this instance. Returns Nil if a Folderitem is not used. Optionally will grant you access to the swap file if one is used.

GetStreamAccess

GetStreamAccess( requireWritable As Boolean = False ) As BinaryStream

Returns a BinaryStream with read access to the data in this instance. Optionally can return a BinaryStream with read/write access to the data in this instance.

ModifyValue

ModifyValue( newValue As BigStringKFS )

Sets the string data in this instance without changing where the data is stored. Allows the BigStringKFS class to be used to easily save data to a specific location.

Operator Overloading

+

Operator_Add(right_operand As BigStringKFS) As BigStringKFS

A BigStringKFS instance can be added to another BigStringKFS, String, MemoryBlock, BinaryStream, or Folderitem instance. Their data is concatenated.

=, <, >, <=, >=, <>

Operator_Compare(other As BigStringKFS) As Integer

A BigStringKFS instance can be compared to another BigStringKFS, String, MemoryBlock, BinaryStream, or Folderitem instance. Their data is compared lexicographically (case-insensitive).

*

Operator_Multiply(iScalar As Integer) As BigStringKFS

Operator_MultiplyRight(iScalar As Integer) As BigStringKFS

A BigStringKFS instance can be multiplied by an integer. The result is <iScalar> instances of the given BigStringKFS instance concatenated together.
