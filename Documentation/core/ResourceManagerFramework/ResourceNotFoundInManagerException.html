ResourceNotFoundInManagerException (Interface)

A ResourceNotFoundInManagerException is raised by a ResourceManagerKFS implementation if the Get method is invoked with a key that does not exist. It is analogous to a KeyNotFoundException for a Dictionary object.

Properties

OffendingKey

OffendingKey As String

Contains the key that was used that caused this exception to be raised.

Compatibility

This class is compatible with Console, Desktop and Web applications, and should work on all platforms.

This class is thread safe.

This class is completely decoupled from the rest of this library.
