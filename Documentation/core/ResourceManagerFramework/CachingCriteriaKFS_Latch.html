CachingCriteriaKFS_AlwaysCache (Implements CachingCriteriaKFS)

The CachingCriteriaKFS_AlwaysCache class can be used when you want to describe when something should be cached. This class is mutable. After a ResourceManagerKFS implementation obtains it, you can change the value of IsCachable to influence behavior.

Constructors

[default]

New CachingCriteriaKFS_Latch

Creates a new CachingCriteriaKFS_Latch object with IsCachable set to True.

Boolean

New SimpleResourceManagerKFS( isCachable As Boolean )

Creates a new CachingCriteriaKFS_Latch object with IsCachable set to the value of the given parameter.

Methods

IsCachable

IsCachable() As Boolean

IsCachable( Assigns isCachable As Boolean )

Returns or sets the value of IsCachable. The default value is True, unless overridden in the constructor.

Compatibility

This class is compatible with Console, Desktop and Web applications, and should work on all platforms.

This class is thread safe.

This class is almost completely decoupled from the rest of this library. All it requires is the CachingCriteriaKFS interface.
