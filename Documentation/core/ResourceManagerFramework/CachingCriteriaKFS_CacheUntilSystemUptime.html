CachingCriteriaKFS_AlwaysCache (Implements CachingCriteriaKFS)

The CachingCriteriaKFS_AlwaysCache class can be used when you want to describe that something should be cached until a specified system uptime.

Constructors

Boolean

New CachingCriteriaKFS_CacheUntilSystemUptime( expirationTimeInMicroseconds As Int64 )

Creates a new CachingCriteriaKFS_CacheUntilSystemUptime object where IsCachable will return True as long as Microseconds is less than the given parameter.

Methods

IsCachable

IsCachable() As Boolean

Returns whether or not the result of the Microseconds method is less than the value provided in the constructor.

Compatibility

This class is compatible with Console, Desktop and Web applications, and should work on all platforms.

This class is thread safe.

This class is almost completely decoupled from the rest of this library. All it requires is the CachingCriteriaKFS interface.
