Resource Manager (Framework)

The Resource Manager Framework is a collection of classes that allow you to define resources at runtime, so that you can use different resources in different situations without needing to recompile or restart with alternate parameters.

Class Interfaces

PersistenceCriteriaKFS

Interface PersistenceCriteriaKFS

A class interface with a single method - IsStillCurrent As Boolean - which returns whether or not the associated data is still up-to-date.

ResourceManagerKFS

Interface ResourceManagerKFS

A class interface that represents an object that, when provided with a String, looks up and returns a Variant based on the given string. This is similar in concept to the Dictionary class, except more restricted in some ways (keys can only be strings) and more flexible in others (it's an interface, not a class, so you can write the implementation almost any way you would like).

Classes

PersistenceCriteriaKFS_AlwaysPersist

Class PersistenceCriteriaKFS_AlwaysPersist

An implementation of the PersistenceCriteriaKFS interface where the IsStillCurrent method always returns True.

PersistenceCriteriaKFS_Latch

Class PersistenceCriteriaKFS_Latch

An implementation of the PersistenceCriteriaKFS interface where the return value of the IsStillCurrent method is set by another method of the class (not part of the interface).

PersistenceCriteriaKFS_NeverPersist

Class PersistenceCriteriaKFS_NeverPersist

An implementation of the PersistenceCriteriaKFS interface where the IsStillCurrent method always returns False.

PersistenceCriteriaKFS_PersistUntilSystemUptime

Class PersistenceCriteriaKFS_PersistUntilSystemUptime

An implementation of the PersistenceCriteriaKFS interface where the IsStillCurrent method returns True as long as the Microseconds method returns a value less than a specified value.

ResourceNotFoundInManagerException

Class ResourceNotFoundInManagerException

The equivalent of the KeyNotFoundException for the Dictionary class.

SimpleResourceManagerKFS

Class SimpleResourceManagerKFS

An implementation of the ResourceManagerKFS interface that only serves the contents of a Dictionary, and is immutable.

Compatibility

This framework is compatible with Console, Desktop and Web applications, and should work on all platforms.

This framework is thread safe.

This framework is completely decoupled from the rest of this library.
