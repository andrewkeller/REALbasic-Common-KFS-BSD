SimpleResourceManagerKFS (Implements ResourceManagerKFS)

SimpleResourceManagerKFS is an immutable implementation of the ResourceManagerKFS interface that only serves the contents of a given Dictionary. The Dictionary is provided to the constructor, and the contents are copied into a private data structure inside the new SimpleResourceManagerKFS object.

Constructors

[default]

New SimpleResourceManagerKFS

Creates a new SimpleResourceManagerKFS object that contains nothing. No key-value pairs can ever be added.

Dictionary

New SimpleResourceManagerKFS( other As Dictionary )

Creates a new SimpleResourceManagerKFS object that contains the key-value pairs of the given Dictionary at the time of running this constructor. After that, key-value pairs cannot be added, deleted, or modified. All of the values are assumed to never expire.

Methods

Fetch

Fetch( key As String, ByRef value As Variant, ByRef cachingCriteria As CachingCriteriaKFS ) As Boolean

Sets the 'value' parameter to the value corresponding to the given key and the 'cachingCriteria' parameter to an object where the IsCachable method always returns True, and returns True.

If the given key does not have a value associated with it, then the 'value' and 'cachingCriteria' parameters are set to Nil and the method returns False.

Get

Get( key As String ) As Variant

Returns the value corresponding to the given key.

If the given key does not have a value associated with it, then a ResourceNotFoundInManagerException is raised.

Lookup

Lookup( key As String, defaultValue As Variant ) As Variant

Returns the value corresponding to the given key.

If the given key does not have a value associated with it, then the 'defaultValue' parameter is returned.

Compatibility

This class is compatible with Console, Desktop and Web applications, and should work on all platforms.

This class is thread safe.

This class is completely decoupled from the rest of this library.
