CachingCriteriaKFS (Interface)

The CachingCriteriaKFS interface can be used when you want an object to be able to describe whether or not some data is still up-to-date. The interface has a single method, IsCachable, which returns Boolean. What you do with that method is completely up to you.

Methods

IsCachable

IsCachable() As Boolean

A method that should return whether or not the object or some data is still up-to-date.

Compatibility

This interface does not impose any restrictions that cause it to be incompatible with any particular type of application (eg Console, Desktop, and/or Web applications).

This interface does not impose any restrictions that cause it to be unsafe in a threadded environment.

This interface is completely decoupled from the rest of this library.
