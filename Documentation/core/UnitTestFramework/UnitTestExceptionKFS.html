UnitTestExceptionKFS (Inherits from RuntimeException)

The UnitTestExceptionKFS method represents a problem with a test case. It could either be a failed assertion, such as "Expected True but found False", or it can be an uncaught exception such as "An unhandled IOException was raised sometime before assertion number 4".

Shared Methods Used as Constructors

NewExceptionFromAssertionFailure

NewExceptionFromAssertionFailure( testClass As UnitTestBaseClassKFS, failure As String, msg As String = "" ) As UnitTestExceptionKFS

NewExceptionFromAssertionFailure( testClass As UnitTestBaseClassKFS, expectedValue As Variant, foundValue As Variant, msg As String = "" ) As UnitTestExceptionKFS

Creates a new UnitTestExceptionKFS that represents the given assertion failure.

NewExceptionFromException

NewExceptionFromException( testClass As UnitTestBaseClassKFS, e As RuntimeException, msg As String = "" ) As UnitTestExceptionKFS

Creates a new UnitTestExceptionKFS that represents the given exception, assuming it was caught. This makes the exception look more like a failed assertion and less like an uncaught exception.

NewExceptionFromUncaughtException

NewExceptionFromUncaughtException( testClass As UnitTestBaseClassKFS, e As RuntimeException, msg As String = "" ) As UnitTestExceptionKFS

Creates a new UnitTestExceptionKFS that represents the given exception, assuming is was not caught.

Methods

GetComponentData

GetComponentData( ByRef scenario As UnitTestArbiterKFS.UnitTestExceptionScenarios, ByRef className As String, ByRef raw_criteria As String, ByRef errNumber As Integer, ByRef expln As String, ByRef assNum As Integer )

Returns through the parameters the component data of the exception. Used by the UnitTestArbiterKFS class when it copies an exception into its internal database.

Shared Methods

FormatCriteria

FormatCriteria( scenario As UnitTestArbiterKFS.UnitTestExceptionScenarios, className As String, raw_criteria As String, errNumber As Integer, assNum As Integer ) As String

Returns a human readable description of the criteria of an exception with the given characteristics. Used by the UnitTestArbiterKFS class to reassemble a criteria description from its component pieces that are stored in its internal database.

FormatMessage

FormatMessage( scenario As UnitTestArbiterKFS.UnitTestExceptionScenarios, className As String, raw_criteria As String, errNumber As Integer, expln As String, assNum As Integer ) As String

Returns a human readable description of the overall message of an exception with the given characteristics. Used by the UnitTestArbiterKFS class to reassemble an exception message from its component pieces that are stored in its internal database.

Properties

AssertionNumber

AssertionNumber() As Integer

Returns the assertion number of the failed assertion, or the assertion number that the exception occurred before. Check the IsAFailedAssertion property to tell which is which.

Criteria

Criteria() As String

Returns the technical reason behind the problem, for example, "Expected 4 but found 2."

ErrorNumber

ErrorNumber() As Integer

Returns the error number of the exception. For assertion failures, the error code is zero.

ExceptionClassType

ExceptionClassType() As String

Returns name of the class of the exception. For assertion failures, "UnitTestExceptionKFS" is returned.

Explanation

Explanation() As String

Returns the textual message at the time of the assertion or exception. Includes the entire message stack.

Message

Message() As String

Returns an explanation of this error. Includes all available data except the stack trace.

ScenarioType

ScenarioType() As UnitTestArbiterKFS.UnitTestExceptionScenarios

Returns the type of failure that this object represents. For example, a failed assertion, a caught exception, and an uncaught exception are all possible scenarios.

Stack

Stack() As String()

Returns the runtime stack trace at the time when this exception was raised.

Compatibility

This class is compatible with console applications, and should work on all platforms.

This class is thread safe.

Notes

When using this class, don't forget that a UnitTestExceptionKFS represents a generic problem. It might be an assertion failure, or it might not. It might be an exception, or it might not. It might be an uncaught exception, or it might not. Don't assume what kind of problem a UnitTestExceptionKFS represents.
