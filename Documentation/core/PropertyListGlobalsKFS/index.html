PropertyListGlobalsKFS (Module)

Provides functionality associated with the PropertyListKFS class that operate with global scope.

Methods

GuessSerializedPListFormat

PropertyListGlobalsKFS.GuessSerializedPListFormat( srcData As BigStringKFS ) As PropertyListKFS.SerialFormats

Guesses the format that the given data is encoded as. Any relevant text encoding should be already set in the BigStringKFS object. This function can detect any of the formats defined in the PropertyListKFS.SerialFormats enumeration.

As suggested by the name of this function, this function does not fully parse the input data. It cannot detect most types of structure errors. It takes a quick glance at the data and guesses which format it probably conforms to. This is completely different from the deserializing operation, which, if the format is not provided, tries every format in order of most restrictive to least restrictive until one of them works.

DeserializePList( srcData As BigStringKFS, fmt As PropertyListKFS.SerialFormats, pgd As ProgressDelegateKFS ) As BigStringKFS

Deserialize the given PropertyListKFS object, and returns the result. The format is determined automatically, however a format can be specified manually. Additionally, a ProgressDelegateKFS object can be provided if the operation might take long enough to want a progress bar.

If srcData is Nil, then it is treated as empty data, which translates to a blank PropertyListKFS object. If the requested format is not supported, then an UnsupportedFormatException is raised. The BigStringKFS class reserves the right to raise an IOException if there is a problem reading the data.

DeserializePListTo

DeserializePListTo( plist As PropertyListKFS, srcData As BigStringKFS, fmt As PropertyListKFS.SerialFormats, pgd As ProgressDelegateKFS, clearFirst As Boolean = True )

Deserialize the given PropertyListKFS object, and stores the result into the given BigStringKFS object. The format is determined automatically, however a can be specified manually. Additionally, a ProgressDelegateKFS object can be provided if the operation might take long enough to want a progress bar.

If srcData is Nil, then it is treated as empty data, which translates to a blank PropertyListKFS object. If plist is Nil, then a NilObjectException is raised. If an error occurs, then some partial data may have been written to the destination. This data is not cleared. If the requested format is not supported, then an UnsupportedFormatException is raised. The BigStringKFS class reserves the right to raise an IOException if there is a problem reading the data.

SerializePList

SerializePList( srcNode As PropertyListKFS ) As BigStringKFS

SerializePList( srcNode As PropertyListKFS, fmt As PropertyListKFS.SerialFormats, pgd As ProgressDelegateKFS ) As BigStringKFS

Serializes the given PropertyListKFS object, and returns the result. By default, the format is PropertyListKFS.SerialFormats.ApplePList, but an alternate can be specified manually. Additionally, a ProgressDelegateKFS object can be provided if the operation might take long enough to want a progress bar.

If srcNode is Nil, then the output data will be non Nil, but empty. If the requested format is not supported, then an UnsupportedFormatException is raised. The BigStringKFS class reserves the right to raise an IOException if there is a problem writing the data.

Compatibility

This module is compatible with console applications, and should work on all platforms.

This module is not thread safe, however internally, it is quite resilient to unexpected data modifications. Although this module will likely not crash, it is still a good idea to lock the objects you're dealing with.
