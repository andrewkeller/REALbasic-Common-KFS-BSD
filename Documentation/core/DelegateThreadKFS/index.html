DelegateThreadKFS (Inherits from Thread)

This class is a Thread that invokes a delegate in the Run event. Used for easily running arbitrary methods in a new thread.

Constructors

[default]

New DelegateThreadKFS

Creates a new DelegateThreadKFS object with a Nil ActionMethod. Basically does nothing, unless you are interested in subclassing.

PlainMethodKFS

New DelegateThreadKFS( d As PlainMethodKFS )

Creates a new DelegateThreadKFS object with the given delegate (PlainMethodKFS). The thread is not set to run immediately.

Notes

This class is rather experimental. If it ends up being useful, then additional delegates will be supported.
