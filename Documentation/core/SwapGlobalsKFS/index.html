SwapGlobalsKFS (Module)

This module aids in allocating and releasing temporary files.

Methods

AcquireSwapFile( [Folderitem], [Boolean] ) As Folderitem

Generates a new swap file, or uses the given suggestion. If you choose to track the file (Boolean), then the usage count of the file is incremented. The file is then returned.

ReleaseAllSwapFiles

Releases all tracked temporary files, regardless of their usage counts. Also clears the tracking pool.

ReleaseSwapFile( Folderitem )

Decrements the usage count of the given file. If the usage count hits zero, the file is deleted and removed from the tracking pool.

Notes

Acquiring the same swap file over and over again increments the usage count of the file. The file must be released an equal number of times in order for it to be deleted. This allows multiple processes to use the same swap file without needing to know whether or not to delete it when they're done.
