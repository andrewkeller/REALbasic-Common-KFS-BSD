ReRaiseRBFrameworkExceptionsKFS

ReRaiseRBFrameworkExceptionsKFS( e As RuntimeException )

Raises the given exception object if and only if it is a EndException or a ThreadEndException.

Intended for use in a Catch-all block, where such an exception might be caught. EndExceptions and ThreadEndExceptions are used in the REALbasic framework, and MUST not be caught and killed by other code.

If there is a possibility that you may have caught one of these exceptions, simply run it through this method. If it really is one of these exceptions, it will be re-raised. If not, then no action will be taken.
