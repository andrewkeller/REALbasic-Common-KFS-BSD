BSDGlobalsKFS_String (Module)

This module contains items that aid with String processing.

Constants

EndOfLineKFS

EndOfLineKFS As String

Same as RB's EndOfLine, but returns a Unix EOL on OS X.

EndOfLineSetKFS

EndOfLineSetKFS As String()

Returns the set of all EndOfLine's. Windows, then Unix, then Mac Classic, in that order. Useful with string utilities that can handle more than one delimiter at a time.

Methods

BinaryStream.InStrB_BSa_KFS

BinaryStream.InStrB_BSa_KFS( ByRef regionStart As UInt64, ByRef regionEnd As UInt64, ByRef matchIndex As Integer, startPos As UInt64, subStrings() As BinaryStream ) As UInt64

Returns the position of the highest priority delimiter in the first clump of delimiters found in the given source string. The region start and end parameters are used to return the position of the start and end of the region where the first clump of delimiters were found, and the matchIndex parameter is used to return the index of the delimiter that was identified as "the match" This delimiter is always inside the delimiter region. Priorities of delimiters are assigned by the order of items in the subStrings array, with the first item being the highest priority.

A common use of this function would be to parse text by lines, without having to know what line endings are used.

Variant.DescriptionKFS

Variant.DescriptionKFS As String

Returns a textual representation of the given Variant for use with sentences. For example, strings are in quotes, numbers are not, and zero, the empty string, True, and False are all written out, and the class name of the object is returned when Variant.StringValue fails.
