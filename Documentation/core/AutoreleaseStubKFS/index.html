AutoreleaseStubKFS (Inherits from Object)

This class is designed to do one simple thing: invoke some delegates when it gets deallocated.

In other words, this class does not deallocate anything itself. It just allows you to do things when it gets deallocated.

Delegates

PlainMethod

Sub PlainMethod()

Note: In REALbasic, if two different delegate declarations have the same arguments and return the same type, then REALbasic treats them as literally the same. For example, you can store a delegate of type AutoreleaseStubKFS.PlainMethod into a variable of type PlainMethodKFS or any other delegate with no arguments and no return type without the need for typecasting. It just fits right in there.

Constructors

[default]

New AutoreleaseStubKFS

Creates a new empty AutoreleaseStubKFS object.

PlainMethod

New AutoreleaseStubKFS( fn As PlainMethod )

Creates a new AutoreleaseStubKFS object that is set to invoke the given PlainMethod delegate.

Methods

Add

Add( fn As PlainMethod )

Adds the given delegate and argument if applicable to the list of delegates to be invoked.

Cancel

Cancel()

Forgets all delegates to be invoked.

InvokeAll

InvokeAll()

Invokes all the delegates that have been acquired thus far in the order that they were added.

Properties

Enabled

Enabled As Boolean

If True, then the InvokeAll method is invoked when this object is deallocated. If False, then no action is taken when this object is deallocated.

Compatibility

This class is compatible with console applications, and should work on all platforms.

This class is thread safe.

This class is completely decoupled from the rest of this library, however the DelegateClosureKFS class greatly enhances the versatility of this class.

Notes

Currently, this class silently catches and destroys all exceptions raised when invoking the delegates. If any of them fail, then it will just move on to the next delegate without telling you in any way.
